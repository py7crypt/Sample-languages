# Sample-languages

## is a python file with list of tuples contains language code and name of it. :shipit: 

###### 1. Requirement:
  - Django  
  ```
    pip install django 
  ```


###### 2. Usage:
  
  - input:
  ```
    activate("ar")
    language = [
            ('af', _('Afrikaans')),
            ('ak', _('Akan')),
            ('sq', _('Albanian')),
            ('am', _('Amharic')),
            ('ar', _('Arabic')),
            ('hy', _('Armenian')),
    ]
    print(language)
   ```
   - output:
    ```
      [
        ('af', 'الإفريقية'), 
        ('fr', 'الفرنسيّة'), 
        ('sq', 'الألبانيّة'), 
        ('ar', 'العربيّة'), 
        ('hy', 'الأرمنية'), 
        ('en', 'الإنجليزيّة')
      ]
    ```
